# client

## Tự động fix các lỗi liên quan Prettier
```bash
yarn run prettier:fix
```
## Tự động fix các lỗi liên quan ESLint
```bash
yarn run lint:fix
```
## Chạy project tại port 3000
```bash
yarn dev
```
## Build project
```bash
yarn run build
```
## Preview kết quả build
```bash
yarn run preview
```
## Kiểm tra lỗi liên quan đến ESLint
```bash
yarn run lint
```
